URL Shortener

A freeCodeCamp Project

Example creation usage:

https://jjurl.herokuapp.com/new/https://www.google.com

https://jjurl.herokuapp.com/new/http://foo.com:8


Example creation output:

{ "originalUrl":"http://foo.com:80", "shortUrl":"https://jjurl.herokuapp.com/817d0" }


Usage:

https://jjurl.herokuapp.com/51bca


Will redirect to:

https://www.google.com/
